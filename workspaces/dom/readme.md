
# Descripition
A tool used to generate di-ui.js's component from lib.dom.ts,
can be imported as a the di-ui.js component

# Usage
`npm i @di-ui.js/dom`
example: 
`import { DrawableAHtmlComponent } from "@di-ui.js/dom"`
to use the `<a></a>` element in di-ui.js


# Build
1. `npm i`
2. `npm run build`
